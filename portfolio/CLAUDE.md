# portfolio/ — the shell

Single-page grid of app cards, vanilla TypeScript + Vite, served at the site
root. Visual spec: `../design/Portfolio Grid.dc.html` (option 1b — uniform
gallery, featured 2×2 card, filter pills).

- Data: `src/generated/manifest.json` — GENERATED by
  `../scripts/build-manifest.mjs` (runs automatically via predev/prebuild).
  Never edit it by hand; edit `apps/*/app.meta.json` or `../external-apps.json`
  instead.
- `src/render.ts` — all DOM rendering. `src/styles.css` — grid styles
  (tokens only, no hardcoded colors).
- Isolation: never open `apps/*/src` from here; at most an app's
  `app.meta.json`.
- Commands: `npm run dev` / `build` / `preview` (or from root: `npm run dev`).
